To run my file for DECISION TREE, please enter the command line arguments as follows:
python filename.py  impurity[1:Entropy 0:Variance] pruning[1:Reduced error pruning] "training_dataset_path" "validation_dataset_path" "testing_dataset_path"
For example:python decision_tree.py 1 1 "C:\\Users\\xyz\\Desktop\\all_data\\train_c300_d5000.csv" "C:\\Users\\xyz\\Desktop\\all_data\\valid_c300_d5000.csv" "C:\\Users\\xyz\\Desktop\\ML data\\\all_data\\test_c300_d5000.csv"
